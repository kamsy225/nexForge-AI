# nexForgeAI

## Personalized Project Recommender for Developers

nexForgeAI helps developers discover their next big project based on their skills, interests, preferred domain, and available time. Powered by TF-IDF content-based filtering and a Skill Gap Analyzer, it delivers smart, tailored project recommendations across multiple tech domains.


## Features

- **Personalized Recommendations**  
  Get top 5 project matches based on your profile.

- **TF-IDF Content-Based Filtering**  
  Uses cosine similarity to rank project relevance.

- **Skill Gap Analyzer**  
  Highlights tools you're missing for each recommended project.

- **Difficulty Score Visualization**  
  Visual progress bar per project to indicate project complexity.

- **Multi-Domain Support**  
  AI/ML, Healthcare, FinTech, Education, Logistics, Web, Mobile, IoT, Hardware, Game Development, and more.

- **Dataset Links**  
  Direct links to relevant datasets for each project.

- **Dark Mode Friendly UI**  
  Automatically adapts to light and dark themes.


## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | Streamlit |
| Recommendation Engine | TF-IDF + Cosine Similarity (scikit-learn) |
| Data Handling | pandas |
| Dataset | Custom `projects.csv` |
| Language | Python 3.x |
