## 📋 What is nexForge-AI?

nexForge-AI is an application designed to help you find your next programming project. It looks at your skills, interests, and how much time you have. Then, it suggests projects that fit you best. It covers many areas like AI, healthcare, fintech, education, logistics, web, mobile, IoT, hardware, and game development.  

The app uses smart methods to rank projects and shows you what skills you might need to learn. It also shows how hard each project is, so you can pick one that fits your level.

## 🛠️ Features Breakdown

- **Personalized Recommendations**  
  The app matches projects to you, not the other way around.

- **Project Ranking**  
  It uses mathematical methods (TF-IDF and cosine similarity) to rank projects by how well they fit your profile.

- **Skill Gap Analyzer**  
  Shows you which skills you still need to learn for each suggested project.

- **Difficulty Score**  
  Visual bar showing how complex each project is; helps you to avoid projects that are too easy or too hard.

- **Wide Domain Support**  
  Recommends projects in various fields such as AI/ML, healthcare, fintech, education, and more.

- **Dataset Links**  
  Provides easy access to data you might need for your project.

- **Recommendations seem off:**

- **App crashes:**  
  Restart your computer and try again. If it still crashes, delete the app and reinstall.
